To register a new menu item:

Decide which template you want to use, and then insert the data into the appropriate db/___.gs file.

Add a function in the appropriate section in menus.gs.

Include an else if function in userproperties.gs

If you want it included in the user bar and navigation, use the function name as the datakey and register it in db/appMap.gs

If it's a "Deep Dive" page, include the function name on the citeKeys spreadsheet.
